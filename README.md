# JX-SHIZUKU

JX-SHIZUKU is an advanced Android management tool designed for **Termux** that utilizes **Shizuku** for privileged access without requiring root. It provides a professional command-line interface to optimize, diagnose, and control your device directly from your terminal.

## Installation

Run the following command in your **Termux** terminal to clone the repository and run the setup script:

```bash
apt install git -y && git clone [https://github.com/mrinosuke/JX-SHIZUKU.git](https://github.com/mrinosuke/JX-SHIZUKU.git) && cd JX-SHIZUKU && chmod +x start.sh && bash start.sh
